# 家族修仙：我的悟性能储存-第81章 道友，别来无恙-69书吧

第81章 道友，别来无恙

广安府港口。

看着一车车灵矿石运送上苍龙号货船，陈道玄皱眉道：“十三叔，你说莫家人会上当吗？”

事到临头，陈道玄难免有些患得患失。

陈家如此苦心布置，若是莫家不来，他们可就竹篮打水一场空了。

陈仙贺则胸有成竹得多，他拍了拍陈道玄的肩膀，笑着道：“放心吧，他们一定会来的。

你还年轻，根本不懂一个年近六十的炼气期修士，对于筑基的渴望！”

陈仙贺还有句话没有说出口。

当初他曾经也这么渴望或者说绝望过。

陈道玄资质惊人，陈家现如今又不缺资源，对他来说，将来筑基是板上钉钉的事情。

他当然不能理解那些无法筑基，或者距离筑基只差临门一脚的人，究竟会有多么绝望、疯狂。

万星海西南海域。

一只船队在碧波荡漾的海面航行。

领头的苍云号，除了装载着陈家购买来的物资外，还装着新一批的出云国女奴。

只不过这次购买的出云国女奴，远没有陈道玄第一次购买的多。

究其原因。

不是陈家没钱买不起，而是广安府年轻的女奴，基本上都快被卖光了。

加上前线的资源运输舰又没来。

广安府这段时间年轻女奴的价格也是居高不下，甚至都快达到两颗灵石一个的恐怖价格。

这么昂贵的价格，陈家自然不愿意再当这个冤大头。

再加上年轻漂亮的女奴基本都被人挑完了，陈道玄此次就只购买了一千余名出云国女奴。

不得不说。

这一年多以来。

双湖岛陈家的普通男性族人，一直处于幸福和痛苦之中。

刚开始，他们觉得幸福，后来就有些痛苦了。

陈道玄不止一次发现，族内的成年男子平常走路时，很多都用手捂着腰走。

两人站在苍云号甲板上。

陈仙贺看了眼身后的船舱，犹豫道：“道玄，我们是不是要暂缓购买出云国女奴，这么下去，我怕族人身体撑不住啊！”

陈仙贺一边抚着胡须，一边劝说道。

闻言，陈道玄瞥了眼船舱，颇为冷酷道：“没关系，我这次为族人购买了大量的补品，不会有事的。”

“.”

听到这话，陈仙贺差点没将胡子给揪断了。

他现在有些庆幸，自己是修士，不是普通的陈氏族人。

就在二人闲谈时。

前方，隐隐有一只巨船朝他们驶了过来。

这艘巨船的速度奇快，只不到半盏茶的功夫，就逼近了陈家的船队。

陈氏叔侄命令船队避让。

孰知对方似乎认准了陈家船队，铁了心朝陈家的船队逼了过来。

看到这一幕。

陈道玄和陈仙贺心中隐隐有了猜测。

两人神识一扫。

果然发现对方船上里有整整二十位炼气期修士。

并且，除了那位黑袍散修外，其他人的修为全都是炼气后期。

换句话，这些莫家修士中，即使修为最差的一个，都达到了炼气七层。

“莫家人怎么跑到我们前面去了？”

陈道玄看向陈仙贺，不解道。

陈仙贺摇了摇头，肃穆道：“修行界追踪之法千奇百怪，他们能在半途等我们，并不奇怪。”

显然。
  
  陈仙贺是见识过一些奇妙追踪之法的。

说着，他又叮嘱道：“待会儿你小心一些，谨防莫家修士狗急跳墙伤到了你。”

闻言，陈道玄抿了抿嘴。

他倒是不担心对方伤到自己，说实话，对于自己现在的实力究竟有多强，陈道玄自己都有些搞不清。

但有一点他能肯定，那就是他的实力放在炼气期修士里面，绝对是近乎无敌的存在。

修行至今。

陈道玄总共只学了两门剑诀，《追风剑诀》和《覆雨剑诀》。

一般而言。

乾元剑宗的外门弟子在炼气期时，必须将《追风剑诀》、《覆雨剑诀》这类一阶剑诀中的一门修炼到大成境界，才算合格。

而内门弟子，除了灵根要求外，则必须将两门以上的一阶剑诀修炼到大成境界。

换句话。

即便乾元剑宗，也不会硬性要求内门弟子将一阶剑诀修炼到圆满境界。

因为这个要求对炼气期弟子来说，太难了。

倘若有哪个内门弟子在炼气期时，就将一阶剑诀修炼到圆满境界，顷刻间就能成为乾元剑宗亲传候补。

只需晋级筑基，领悟剑意，便能位列亲传弟子。

广安府的周慕白，若是出身在乾元剑宗，应当就能成为一名亲传弟子。

而陈道玄，三个月就将《追风剑诀》修炼圆满，而后又花三个月，将《覆雨剑诀》修炼圆满。

之后整整四个月，卡在《追风剑诀》和《覆雨剑诀》的融合上。

直到一年前观海亭悟道。

领悟了《追风覆雨剑诀》的奥义，从此剑道水平进入了新的天地。

自此之后。

整整一年时间。

陈道玄每日都在双湖岛潜修，每十日顿悟一次，每次顿悟，剑道水平都有所精进。

一年时间顿悟下来，陈道玄也不知道自己现在的剑道境界，究竟达到了何种地步。

他只知道，现在的自己若是对上一年前的自己，一剑就能秒杀。

他隐隐已经感知到，自己的剑道境界，又处于一道新的门槛前。

若是跨过去，将是一片全新的天地。

奈何在这方面。

陈家没有任何先辈留下只言片语可以指导他。

他甚至不知道剑道境界到底该如何划分。

他只知道周慕白在筑基期时领悟了剑意，成为了一名剑修。

剑意，应该就是剑道境界的一种描述。

陈道玄不知道自己有没有领悟剑意，算不算是剑修。

不过根据传闻，他应该还没有领悟剑意才对。

因为传说中，修士在领悟剑意后，会对修士的真元进行一番彻底的淬炼，将真元转化为无坚不摧的剑元。

这也是剑修同阶攻伐第一，甚至可以越阶而战的由来。

沉思间。

对方的船只已经在陈家船队前不远处停了下来。

见状。

陈家也不得不停下船队，静候在此。

下一刻。

二十位修士冲天而起，御风而行，朝着陈家船队飞了过来。

为首的那位，正是二十天前，两人才照过面的莫家族长，莫长生。

此刻。

莫长生正居高临下的看着陈氏叔侄，嘴角噙着一抹冷笑，道：“数日不见，两位道友别来无恙？”

(本章完)
